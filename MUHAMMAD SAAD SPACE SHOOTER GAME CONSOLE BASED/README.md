SPACE SHOOTER CONSOLE BASED GAME

OverView:

1.A space shooter console based game made using C++ and SFML for GUI
2.Used to kill the enemies who come and fire at you.
3.User friendly console based game with score updates, restart, save game, pause options.

Features:

1.Allowing user to shoot at enemy.
2.Allowing to reach various levels with increasing difficulties.
3.Allowing to save a game and load it later.
4.Background music

Technologies used:

1.C++
2.Attractive Console Based User interface 
3.FileHandling

Project Structure:

1.Main calls all the functions 
2.file handling after save or load is used.
3.Several other supporting functions are used as well to create enemy, bullet, movement.

Running procedure:

1.Clone following github repository:
https://github.com/MuhammadSaad777/SPACE-SHOOTER-GAME-CONSOLE-BASED

2.Open project in VisualStudio

3.Compile project and run it
